## âš¡ TROUBLESHOOTING GUIDE

### Problem: Border not visible

**Possible causes and fixes:**
1. Border colour same as surrounding background
   - Fix: Use #3C0920 as specified
2. Border width too thin
   - Fix: Increase to 2px or 3px
3. Border style not set
   - Fix: Ensure style is 'solid'
4. CSS specificity issue
   - Fix: Add `!important` or increase specificity

### Problem: Background not visible

**Possible causes and fixes:**
1. Background same as page background
   - Fix: Choose different colour from palette
2. Parent element overriding
   - Fix: Check parent styles, add `!important` if needed
3. Wrong element styled
   - Fix: Ensure you're styling the container element

### Problem: Section looks cramped

**Fix:** Increase padding values
```
From: padding: 12px
To: padding: 24px or padding: 32px
```

### Problem: Text not readable

**Fix:** Change background colour choice
- If using #44062D (dark), text must be light (#F7F2EF)
- If using #EED1C4 or #F7F2EF (light), text should be dark (#44062D or #3C0920)

### Problem: Styling not applying

**Debug steps:**
1. Check spelling of class names
2. Check file was saved
3. Clear browser cache
4. Check CSS specificity (use DevTools)
5. Try inline styles as fallback
