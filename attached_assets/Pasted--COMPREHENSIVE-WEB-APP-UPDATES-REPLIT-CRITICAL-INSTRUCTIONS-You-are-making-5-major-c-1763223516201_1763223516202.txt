# üîç DEBUGGING GUIDE

## If you encounter errors:

### Console Error: "Cannot read property of undefined"
**Cause:** Trying to access booking data that doesn't exist  
**Fix:** Add null checks before accessing properties
```javascript
const message = bookingDetails?.serviceName || 'Service';
```

### Visual Issue: Spacing looks different on mobile
**Cause:** Not using responsive classes  
**Fix:** Add responsive Tailwind classes
```jsx
className="p-4 md:p-6 lg:p-8"
```

### WhatsApp doesn't open
**Cause:** Invalid URL encoding or popup blocker  
**Fix:** 
1. Check encodeURIComponent is used correctly
2. Add user interaction before window.open
3. Test in incognito mode (no extensions)

### Gradient looks banded/striped
**Cause:** Not enough gradient stops  
**Fix:** Add intermediate colour stops
```css
background: linear-gradient(to bottom, 
  #44062D 0%, 
  #3C0920 25%,
  #5C4557 50%,
  #9B7B88 75%,
  #EED1C4 100%
);
```

### Text not readable on gradient
**Cause:** Contrast too low  
**Fix:** 
1. Adjust gradient colours
2. Split content into dark and light sections
3. Add text shadow for better readability
```css
text-shadow: 0 1px 2px rgba(0,0,0,0.5);
```

### Background doesn't extend full width
**Cause:** Parent container has max-width  
**Fix:**
```jsx
<div className="w-full bg-[#EED1C4]">
  <div className="max-w-7xl mx-auto">
    {/* Content */}
  </div>
</div>
```
