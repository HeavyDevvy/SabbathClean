Or ../assets/berry-logo.png (if in src folder)




STEP 2: Create or Update Logo Component
Create a reusable BerryLogo component:
javascript// File: src/components/BerryLogo.jsx (or add to your components folder)

import React from 'react';

export const BerryLogo = ({ size = 'medium', className = '' }) => {
  // Define logo sizes
  const sizes = {
    small: { width: '80px', height: '80px' },     // For footer or small spaces
    medium: { width: '120px', height: '120px' },   // For standard use
    large: { width: '180px', height: '180px' },    // For HEADER - large & noticeable
  };

  return (
    <img
      src="/assets/berry-logo.png"  // UPDATE THIS PATH to match where you saved the logo
      alt="Berry Events"
      className={`berry-logo ${className}`}
      style={{
        width: sizes[size].width,
        height: sizes[size].height,
        objectFit: 'contain',
      }}
      onError={(e) => {
        console.error('Logo failed to load. Check the file path.');
        e.target.style.display = 'none'; // Hide if image fails to load
      }}
    />
  );
};

TROUBLESHOOTING
If logo doesn't appear:

Check file path is correct: /assets/berry-logo.png
Verify file exists in the correct folder
Check console for 404 errors
Try absolute path: https://your-replit-url.com/assets/berry-logo.png

If icons still wrong color:

Check for CSS overrides using browser inspector
Add !important to color styles if needed
Verify you're using correct icon components (lucide-react)

If header layout breaks:

Reduce logo size slightly: change 180px to 150px
Adjust header padding
Check flex layout properties
